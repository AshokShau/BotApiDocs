Connect a New Bot

Go to @botfather and create a /newbot to obtain this.

Setup

  1. Go to BotFather and create a new bot or choose an existing one.
  2. Copy the bot token and paste it into the input above.
  3. Click connect and your bot is all set up.

Usage

  1. Start a chat with your bot on Telegram. Once the bot is running, you can search for API methods and types..
  2. Use the inline query feature by typing @YourBotUsername your_query to search for methods or types.
  3. You're all set!
